## Scope

Enables extraction of React component rendering trees by parsing JSX markup tags as dependency edges.

1. **F35 — JSX Component Rendering Edges** (🔴 CRITICAL):
   - Updates references parser queries for JS/TS/TSX files to capture opening and self-closing JSX elements (e.g. `<LinksPage />`, `<DashboardLayout>`).
   - Inserts these references as dependency edges (either labeled as `render` or `call`) between the parent component/file and the target component symbol.
