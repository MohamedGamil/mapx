## Scope

Every built-in and bundled language must fully capture all of its syntactic constructs for **symbols** (classes, functions, methods, interfaces, enums, structs, modules, constants, properties, traits) and **references** (imports, calls, instantiations, inheritance, implementations). PHP serves as the baseline reference at ~9 symbol kinds and ~109 lines of reference queries.

## Language-Specific Requirements

### Python (built-in)
**Symbols**: Add decorator, constant (module-level UPPER_CASE assignments), property (`@property` methods), method (differentiate from top-level function by class scope)
**References**: Add instantiation (`ClassName()`), method calls (`obj.method()`), decorator references (`@decorator`)
**Registry**: Add `constant`, `property`, `method` mappings

### Go (built-in)
**Symbols**: Add constant (`const_spec`), type alias (`type_spec` non-struct/interface), interface method signatures
**References**: Add `extends` equivalent for interface embedding
**Registry**: Add `constant` mapping

### Rust (built-in)
**Symbols**: Add `impl_item` blocks, `const_item`, `static_item`, `type_item` alias, macro definitions (`macro_definition`, `macro_rules!`)
**References**: Add trait implementations (`impl Trait for Type`)
**Registry**: Add `constant`, `module`, `property` (struct fields)

### Java (built-in)
**Symbols**: Add annotation type, field declarations, constants (`static final`)
**References**: Add `extends`, `implements`, generic type parameters
**Registry**: Add `constant`, `property` (fields), `namespace` (package)

### C# (built-in)
**Symbols**: Add property declarations, constants, namespace, delegate, record, event
**References**: Add `extends` (base class), `implements` (interface), generic type parameters
**Registry**: Add `constant`, `property`, `namespace`

### Ruby (bundled)
**Symbols**: Add module, constant, `attr_reader`/`attr_writer`/`attr_accessor` as properties, singleton methods
**References**: Add `require`, `require_relative`, `include`/`extend`/`prepend`, inheritance (`< SuperClass`), instantiation (`.new`)
**Registry**: Add `module`, `constant`, `property`

### C (bundled)
**Symbols**: Add enum (`enum_specifier`), typedef (`type_definition`), macro definitions (`preproc_def`/`preproc_function_def`), union
**References**: Add `#include` directives
**Registry**: Add `enum`, `constant`

### C++ (bundled)
**Symbols**: Add namespace (`namespace_definition`), enum (`enum_specifier`), template class/function, virtual/override methods
**References**: Add `#include`, inheritance (`: public Base`), instantiation (`new ClassName`)
**Registry**: Add `namespace`, `enum`, `method`

### Swift (bundled)
**Symbols**: Add protocol (`protocol_declaration`), enum (`enum_declaration`), extension (`extension_declaration`), property (`variable_declaration` at class scope), initializer
**References**: Add import, conformance (`: Protocol`), inheritance (`: SuperClass`)
**Registry**: Add `interface` (protocol), `enum`, `property`

### Kotlin (bundled)
**Symbols**: Add interface (`interface_declaration`), object (`object_declaration`), enum (`enum_class_declaration`), data class, property (`property_declaration`), companion object
**References**: Add import, inheritance (`: SuperClass`), implementation (`: Interface`), instantiation
**Registry**: Add `interface`, `enum`, `property`, `constant`

### Dart (bundled)
**Symbols**: Add enum (`enum_declaration`), mixin (`mixin_declaration`), extension (`extension_declaration`), const/final top-level
**References**: Add extends, implements, with (mixin), instantiation
**Registry**: Add `enum`, `constant`, `interface` (abstract class)

### Scala (bundled)
**Symbols**: Add trait (`trait_definition`), case class, val/var declarations, package object
**References**: Add extends, with (mixin), instantiation
**Registry**: Add `interface` (trait), `constant` (val)

### Vue (bundled)
**Symbols**: Add component (`defineComponent`/`<script setup>`), props, computed properties, composable functions, emits
**References**: Add import (from `<script>` block), component references
**Registry**: Add `class` (component), `property` (props/computed)

## Acceptance Criteria

- [ ] Every built-in language (PHP, JS, TS, Python, Go, Rust, Java, C#) captures ≥6 symbol kinds
- [ ] Every bundled language (Ruby, C, C++, Swift, Kotlin, Dart, Scala, Vue) captures ≥4 symbol kinds
- [ ] Every language captures imports/requires where applicable
- [ ] Every language captures inheritance/implementation where applicable
- [ ] Every language captures instantiation where applicable
- [ ] `nodeMappings` in `registry.ts` matches all captured symbol kinds
- [ ] TypeScript compiles with 0 errors
- [ ] Scanning a representative test file per language extracts expected symbols

## Implementation Order

1. **Python** — highest usage, most critical gap (built-in with only 2 symbol kinds)
2. **Ruby** — second most critical (bundled, 3 symbols, no imports)
3. **Kotlin** — very minimal (2 symbols, no imports)
4. **C / C++** — critical system languages with minimal coverage
5. **Swift** — missing protocol, enum, import
6. **Java / C#** — good base but missing extends/implements, fields, constants
7. **Go / Rust** — decent base, incremental improvements
8. **Dart / Scala** — moderate gaps
9. **Vue** — framework-specific, most complex due to SFC structure
