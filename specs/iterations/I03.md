## Objective

Expose two new user-facing commands and their MCP equivalents:

- **`mapx metrics`** / **`mapx_metrics`** — per-file coupling report (in-degree, out-degree, instability, PageRank)
- **`mapx edges`** / **`mapx_edges`** — structured, filterable edge neighbourhood for a single file

F04 (`edges`) is independent of I01. F02 (`metrics`) can be started independently, but the `--verified-only` flag requires I01 to be merged first.
